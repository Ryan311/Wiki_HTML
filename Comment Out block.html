Commnet Out for Block

Comment block for program langurage

Python

  • Add this two line in Vimrc
    vnoremap # :norm 0i#<cr>
    vnoremap -# :norm 0x<cr>
  • Usage
    • Visual Mode, use Shift-v to select a block
    • press #, then the block is commentted out by '#'
  • Detail
    • Shift-v select a block
    • :norm deal with each line
    • 0i#
      0 goto the start of the line
      i change into insert mode
      # insert a '#'

C

vnoremap / :norm i0//<cr> vonremap -/ :norm 02x<cr>

vim script

vnoremap " :norm i0"<cr> vnoremap -" :norm 0x<cr>
