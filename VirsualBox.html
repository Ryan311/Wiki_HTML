Share fold with Host

  1. In Host, add /usr/share/virsualbox/VBoxGuestAdditions.iso to Settings->Storage->CD/DVD
  2. In Guest(Ubuntu12.04 Server), need to install build-essential and dkms first.
    sudo apt-get update
    sudo apt-get install build-essential
  3. In Guest, mount CD.
    cd /media
    sudo mkdir vbox
    sudo mount /dev/sr0 vbox/
    cd vbox/
  4. In Guest
    sudo ./VBoxLinuxAdditions.run
  5. Add Share Folder in Settings->Shared Folders->Add
  6. Restart Guest, then
    sudo mount -t vboxsf Host /mnt
