Bugs

  • 单目运算符的优先级很高， 且结合方向是自右向左， 
            char p[10] = "1234567890";
        int *a = (int *)p++;
        int *b = (int *)p++;
        
        执行完后， p是指向p[3]的，因为p++先执行， 然后才会强制转换
        要实现p向后移4 byte， 需要先将p强制转换成int *的
        ((int *)p)++;           这个是语法错误
        int *p1 = (int *)p;     这样OK
