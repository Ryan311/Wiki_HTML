  • Service Record表示一个服务， 由Record ID区分不同的服务实例， Attribute ID(0)表示Record ID
  • Service Class，预先定义的服务声明，它规定了属于该类的服务要支持哪些属性，每个Record都是Class的一个实例
  • Service Attribute: 由Attribute ID和Attribute Value组成， 由Data Element形式表示
  • Request/Response Model
    • Service Search Request/Response 通过UUID查看Service的Record Handle（Attribute ID为0）
    • Service Attribute Request/Response 已知Service的Record Handle， 查找该服务下的Attributes， 可以指定一个AttID， 也可是一个范围
    • Service Search Attribute Request/Response 上面二者的结合，指定UUID查找服务， 并返回该服务对应的Attributes
  • Continuation State
    • This is generated by SDP server. When the record cannot be transferred once, this state should be set
    • to nozero. SDP Client should send another request with the continuation state same to last response
  • Data Representation : Data Element 
            Header Field
            Type Descriptor
                specify length      (high-order) 5 bits
            Size Descriptor 
                specify meaning     (low-order) 3-bit size index followed by 0, 8, 16, or 32 bits
